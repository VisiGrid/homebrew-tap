# Homebrew Tap for VisiGrid

This is the official [Homebrew](https://brew.sh) tap for [VisiGrid](https://visigrid.app), a fast, native spreadsheet with GPU-accelerated rendering.

## Installation

### macOS

```bash
brew tap visigrid/visigrid
brew install --cask visigrid
```

### Linux

```bash
brew tap visigrid/visigrid
brew install visigrid
```

## Updating

### macOS

```bash
brew update
brew upgrade --cask visigrid
```

### Linux

```bash
brew update
brew upgrade visigrid
```

## Uninstalling

### macOS

```bash
brew uninstall --cask visigrid
brew untap visigrid/visigrid
```

### Linux

```bash
brew uninstall visigrid
brew untap visigrid/visigrid
```

## Requirements

### macOS
- macOS 12.0 (Monterey) or later
- Apple Silicon (M1/M2/M3) or Intel Mac (Universal binary)

### Linux
- x86_64 architecture
- GTK 3, libxcb, libxkbcommon
